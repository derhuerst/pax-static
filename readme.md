## Installation

```shell
npm install pax-static
```


## Usage

Returns the path of the `pax` binary on the local filesystem.

``` js
const pathToPax = require('pax-static-macos')
console.log(pathToPax)
// /Users/j/playground/node_modules/pax-static-macos/pax
```
